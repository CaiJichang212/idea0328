# 营养健康管家 Web应用 前端设计需求文档

## 文档说明

本文档为Figma AI设计工具提供完整的前端界面设计规范，包含色彩系统、排版样式、组件规范、页面布局、交互行为等全部设计元素。所有设计参数均提供具体数值，确保AI能够准确生成符合需求的设计稿。

---

## 1. 设计系统基础

### 1.1 设计理念

| 维度 | 描述 |
|-----|------|
| **整体风格** | 简洁、现代、健康、专业 |
| **设计语言** | 扁平化设计，微质感，柔和圆角 |
| **视觉层次** | 清晰的信息层级，突出核心数据 |
| **情感基调** | 积极、可信赖、易亲近 |

| **space-3** | 12px | 标准元素间距 |
| **space-4** | 16px | 常规间距、卡片内边距 |
| **space-5** | 20px | 区块间距 |
| **space-6** | 24px | 大间距、区块外边距 |
| **space-8** | 32px | 区块间距 |
| **space-10** | 40px | 页面内边距 |
| **space-12** | 48px | 大区块间距 |

## 2. 组件库规范

| 类型 | 样式 | 使用场景 |

| **Large** | 48px | 20px 32px | 16px | 12px |

## 4. 交互行为规范

### 4.1 按钮交互

| 状态 | 视觉反馈 | 动画时长 |
|-----|---------|---------|
| **Hover** | 背景色加深10% | 150ms |
| **Pressed** | 背景色加深20%，轻微缩放(0.98) | 100ms |
| **Disabled** | 灰色显示，无交互 | - |
| **Loading** | 显示旋转加载图标 | - |

### 4.2 输入框交互

| 状态 | 视觉反馈 |
|-----|---------|
| **Focus** | 边框变为主色，显示外发光阴影(0 0 0 3px rgba(34,197,94,0.2)) |
| **Error** | 边框变为红色，下方显示错误提示 |
| **Disabled** | 背景变灰，禁止输入 |

### 4.3 列表项交互

| 状态 | 视觉反馈 |
|-----|---------|
| **Hover** | 背景 #F9FAFB |
| **Pressed** | 背景 #F3F4F6 |
| **Swipe Left** | 显示删除按钮（移动端） |

### 4.4 卡片交互

| 状态 | 视觉反馈 |
|-----|---------|
| **Hover** | 阴影加深，轻微上移(translateY: -2px) |
| **Pressed** | 阴影减弱，轻微下移 |

### 4.5 页面转场动画

| 转场类型 | 动画效果 | 时长 |
|---------|---------|------|
| **页面进入** | 从右侧滑入 | 300ms |
| **页面退出** | 向右滑出 | 300ms |
| **弹窗显示** | 从中心缩放进入 | 200ms |
| **弹窗关闭** | 向中心缩放消失 | 150ms |
| **底部弹窗** | 从底部滑入 | 300ms |

### 4.6 手势操作（移动端）

| 手势 | 操作 | 使用场景 |
|-----|------|---------|
| **下拉** | 刷新页面 | 首页、列表页 |
| **左滑** | 显示删除按钮 | 列表项 |
| **右滑** | 返回上一页 | 全局 |
| **长按** | 显示操作菜单 | 列表项、图片 |
| **双指缩放** | 放大/缩小图片 | 图片查看 |

### 4.7 加载状态

| 类型 | 样式 | 使用场景 |
|-----|------|---------|
| **页面加载** | 全屏加载动画 + 提示文字 | 页面初始加载 |
| **区块加载** | 骨架屏(Skeleton) | 内容区块加载 |
| **按钮加载** | 按钮内旋转图标 | 表单提交 |
| **下拉刷新** | 顶部旋转图标 | 列表刷新 |
